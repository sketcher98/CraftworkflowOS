## Playbook Categories

### Outreach Playbooks
- **DM_Flows.md** - 5 distinct DM flows with specific use cases, openers, and follow-up sequences
- **Daily_Targeting_Playbook.md** - 45-75 min per day, step-by-step LinkedIn/Twitter routine
- **Lead_Sourcing.md** - Where to find leads (4 categories + who to avoid)
- **Lead_Qualification.md** - Buying signals, pain signals, disqualifiers

### Sales Playbooks
- **Sales_Call_Playbook.md** - Complete call script with frame control, discovery, cost, reframe, offer, close
- **Pre_Call_Sequence.md** - 4-email pre-call sequence + SMS nudge
- **Post_Call_Followup.md** - Post-call recap email, returning prospect script
- **Objection_Handling.md** - 8 objections with exact responses
- **Roleplay_Scenarios.md** - 8 roleplays: hostile, DIY, CTO, profitable, proposal, burned, risky, bad call recovery

### Marketing Playbooks
- **Content_Pillars.md** - 4 pillars: Belief, Parable, Proof, Process (Goat Farmer edition)
- **Content_Hooks.md** - Hook templates for Twitter/LinkedIn
- **Platform_Strategy.md** - Twitter (beliefs/parables) + LinkedIn (long-form/case studies)
- **Funnel_Structure.md** - 5-step funnel: Traffic → Audit → Jumpstart → OS → Optimization

### Operations Playbooks
- **Daily_Operations.md** - Daily startup/shutdown routine
- **Weekly_Review.md** - Weekly review template
- **Project_Kickoff.md** - Project initiation checklist

### Delivery Playbooks
- **Client_Onboarding.md** - Onboarding workflow
- **Project_Kickoff.md** - Project initiation
- **Quality_Assurance.md** - QA checklist

### Finance Playbooks
- **Pricing_Packages.md** - Jumpstart ($2.5-4k), Goldilocks ($6-12k), Visionary ($15-30k)
- **Proposal_Template.md** - Proposal structure
- **Invoice_Template.md** - Invoice format

## Usage Guidelines

1. **Reference don't duplicate** - Link to documents rather than copying content
2. **Version control** - Update dates on modifications
3. **Cross-reference** - Link related playbooks
4. **Test before documenting** - Only document what works
