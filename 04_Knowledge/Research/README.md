## Research Standards

1. **Source every claim** - Link to source or note "internal observation"
2. **Date everything** - Research goes stale fast
3. **Separate signal from noise** - Distinguish trends from hype
4. **Action-oriented** - Every piece should inform a decision

## Research Workflow

1. **Capture** - Save raw findings to Inbox
2. **Process** - Extract insights, file in appropriate folder
3. **Synthesize** - Update strategy documents
4. **Act** - Create tasks, update playbooks
5. **Archive** - Move to long-term storage
